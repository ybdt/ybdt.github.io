0x01 目标熟悉

JPress 是一个使用 Java 开发的、开源免费的建站神器，灵感来源于 WordPress，目前已经有超过 10w+ 的网站使用 JPress 搭建，其中包括多个政府机构，200+上市公司，中科院、红十字会等。

比起普通的 SpringBoot 搭建的 Java 环境相比，jpress 使用和 SpringBoot 十分类似的 jBoot
