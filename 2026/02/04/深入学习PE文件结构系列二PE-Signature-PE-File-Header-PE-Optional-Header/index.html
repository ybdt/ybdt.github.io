ж·ұе…ҘеӯҰд№ PEж–Үд»¶з»“жһ„зі»еҲ—дәҢPE-Signature-PE-File-Header-PE-Optional-Header

Created2026-02-04|Updated2026-02-04|Windows PE
|Post Views:

0x1 еүҚиЁҖ

дёҠдёҖзҜҮж–Үз« жҺўи®ЁдәҶDOS Headerд»ҘеҸҠйҖҶеҗ‘дәҶDOS StubпјҢжң¬зҜҮж–Үз« и°Ҳи®әNT Headers

ејҖе§Ӣд№ӢеүҚжҲ‘д»¬йңҖиҰҒе…ҲжҳҺзЎ®дёҖдёӘйҮҚиҰҒжҰӮеҝөпјҢRVAпјҲRelative Virtual AddressпјҢзӣёеҜ№иҷҡжӢҹең°еқҖпјүпјҢе®ғжҳҜзӣёеҜ№дәҺеҹәеқҖзҡ„еҒҸз§»ең°еқҖпјҢе…¶дёӯжҳ еғҸеҹәеқҖжҳҜPEж–Үд»¶иў«иҪҪе…ҘеҶ…еӯҳеҗҺзҡ„иө·е§Ӣең°еқҖпјҢд№ҹе°ұжҳҜиҜҙе®һйҷ…ең°еқҖ = зӣёеҜ№иҷҡжӢҹең°еқҖ + жҳ еғҸеҹәеқҖ

еӣһйЎҫдёҖдёӢдёҠдёҖзҜҮж–Үз« дёӯжҸҗеҲ°зҡ„пјҢPEж–Үд»¶з»“жһ„д»ҺдёҠеҲ°дёӢдҫқж¬ЎжҳҜпјҡDOS Header -> DOS Stub -> Rich Header -> NT Headers -> Section Header -> Section1 Section2 вҖҰ SectionN

е…¶дёӯпјҢNT Headersд»ҺдёҠеҲ°дёӢдҫқж¬ЎжҳҜпјҡPE Signature -> PE File Header -> PE Optional HeaderпјҢжң¬зҜҮж–Үз« е°ұжҳҜиҜҰз»Ҷи®Іиҝ°иҝҷйғЁеҲҶеҶ…е®№

0x2 NT Headers

NT HeadersжҳҜдёҖдёӘз»“жһ„дҪ“пјҢеҗҚдёәIMAGE_NT_HEADERSпјҢз»“жһ„дҪ“е®ҡд№үдҪҚдәҺwinnt.hдёӯпјҢе®ҡд№үеҰӮдёӢ

_IMAGE_NT_HEADERS64жҳҜй’ҲеҜ№64дҪҚPEпјҢ_IMAGE_NT_HEADERSжҳҜй’ҲеҜ№32дҪҚPEпјҢеҸҜд»ҘзңӢеҲ°пјҢе®ғжңү3дёӘжҲҗе‘ҳпјҡ

  • DWORDзұ»еһӢзҡ„Signature
  • IMAGE_FILE_HEADERз»“жһ„дҪ“зұ»еһӢзҡ„FileHeader
  • IMAGE_OPTIONAL_HEADER64(IMAGE_OPTIONAL_HEADER32)з»“жһ„дҪ“зұ»еһӢзҡ„OptionalHeader

0x3 PE Signature

NT Headersдёӯзҡ„з¬¬1дёӘжҲҗе‘ҳпјҢеҸҳйҮҸеҗҚжҳҜSignatureпјҢDWORDзұ»еһӢиЎЁзӨәе®ғжҳҜдёҖдёӘ4еӯ—иҠӮж— з¬ҰеҸ·ж•ҙеһӢпјҢе®ғйҮҢйқўжҳҜдёҖдёӘеӣәе®ҡеҖј0x00004550пјҢx86жһ¶жһ„дҪҝз”Ёе°Ҹз«ҜеәҸпјҢжүҖд»ҘеңЁеҶ…еӯҳдёӯзҡ„йЎәеәҸдёә50 45 00 00пјҢзҝ»иҜ‘жҲҗASCIIжҳҜP E \0 \0пјҢжҲ‘д»¬з”ЁPE-bearжҹҘзңӢдёҖдёӘе®һйҷ…PEж–Үд»¶

0x4 PE File Header

NT Headersдёӯзҡ„з¬¬2дёӘжҲҗе‘ҳпјҢд№ҹеҸ«еҒҡCOFF File HeaderпјҢжҳҜдёҖдёӘIMAGE_FILE_HEADERзұ»еһӢзҡ„з»“жһ„дҪ“пјҢз»“жһ„дҪ“е®ҡд№үдҪҚдәҺwinnt.hдёӯпјҢе®ҡд№үеҰӮдёӢ

е®ғеҢ…еҗ«7дёӘжҲҗе‘ҳ

  • MachineпјҡиҝҷдёӘеӯ—ж®өиЎЁзӨәеҸҜжү§иЎҢж–Үд»¶жҳҜиҝҗиЎҢеңЁx86иҝҳжҳҜx86-64дёҠпјҢеҖјдёә0x14cиЎЁзӨәx86пјҢеҖјдёә0x8864иЎЁзӨәx86-64пјҢиҝҳжңүе…¶д»–еҖјпјҢдҪҶжҳҜжҲ‘д»¬дёҚе…іеҝғпјҢе…·дҪ“еҸҜд»ҘеҸӮиҖғпјҡhttps://learn.microsoft.com/en-us/windows/win32/debug/pe-format
  • NumberOfSectionsпјҡиҝҷдёӘеӯ—ж®өиЎЁзӨәиҠӮзҡ„ж•°йҮҸпјҲиҠӮзҡ„ж•°йҮҸд№ҹе°ұжҳҜиҠӮеӨҙзҡ„ж•°йҮҸпјү
  • TimeDateStampпјҡиҝҷдёӘеӯ—ж®өжҳҜдёҖдёӘunixж—¶й—ҙжҲіпјҢиЎЁжҳҺж–Үд»¶иў«еҲӣе»әзҡ„ж—¶й—ҙ
  • PointerToSymbolTableе’ҢNumberOfSymbolsпјҡеҪ“жңүCOFFз¬ҰеҸ·иЎЁзҡ„ж—¶еҖҷпјҢPointerToSymbolTableиЎЁзӨәзӣёеҜ№COFFз¬ҰеҸ·иЎЁзҡ„ж–Үд»¶еҒҸз§»пјҢNumberOfSymbolsиЎЁзӨәз¬ҰеҸ·иЎЁдёӯйЎ№зҡ„ж•°йҮҸпјҢдҪҶжҳҜз”ұдәҺдёҚйј“еҠұдҪҝз”ЁCOFFи°ғиҜ•дҝЎжҒҜпјҢжүҖд»Ҙе№¶жІЎжңүCOFFз¬ҰеҸ·иЎЁпјҢиҝҷдёӨйЎ№йҖҡеёёиў«и®ҫзҪ®дёә0
  • SizeOfOptionalHeaderпјҡиҝҷдёӘеӯ—ж®өиЎЁзӨәPE Optional Headerзҡ„еӨ§е°Ҹ
  • CharacteristicsпјҡиЎЁжҳҺж–Үд»¶еұһжҖ§зҡ„ж Үеҝ—пјҢж–Үд»¶еҸҜд»ҘжҳҜеҸҜжү§иЎҢж–Үд»¶пјҢд№ҹеҸҜд»ҘжҳҜзі»з»ҹж–Үд»¶пјҢзӯүзӯүпјҢе…·дҪ“еҸҜд»ҘеҸӮиҖғпјҡhttps://learn.microsoft.com/en-us/windows/win32/debug/pe-format

дёӢеӣҫжҳҜдёҖдёӘе®һйҷ…PEж–Үд»¶зҡ„PE File HeaderйғЁеҲҶ

0x5 PE Optional Header

NT Headersдёӯзҡ„з¬¬3дёӘжҲҗе‘ҳпјҢд№ҹжҳҜдёҖдёӘз»“жһ„дҪ“пјҢе®ғеҸ«PEеҸҜйҖүеӨҙжҳҜеӣ дёәжңүдәӣж–Үд»¶зұ»еһӢпјҢжҜ”еҰӮеҜ№иұЎж–Үд»¶е№¶жІЎжңүиҝҷйғЁеҲҶпјҢдҪҶеҜ№PEж–Үд»¶жқҘиҜҙпјҢе®ғжҳҜиҮіе…ійҮҚиҰҒзҡ„пјҢе®ғжІЎжңүдёҖдёӘеӣәе®ҡеӨ§е°ҸпјҢеҸҜд»Ҙд»ҺPE File Headerзҡ„жҲҗе‘ҳSizeOfOptionalHeaderиҺ·еҸ–PE Optional Headerзҡ„еӨ§е°ҸпјҢз»“жһ„дҪ“е®ҡд№үеҰӮдёӢ

еҸҜд»ҘзңӢеҲ°пјҢз»“жһ„дҪ“зҡ„еүҚ8дёӘжҲҗе‘ҳжҳҜCOFFж–Үд»¶ж јејҸзҡ„ж ҮеҮҶжҲҗе‘ҳпјҲжіЁж„ҸPE32жҳҜ9дёӘжҲҗе‘ҳпјҢPE32+жҳҜ8дёӘжҲҗе‘ҳпјүпјҢеү©дёӢзҡ„жҲҗе‘ҳжҳҜеҫ®иҪҜй’ҲеҜ№Windows PEжү©еұ•зҡ„

е°ұеғҸNT HeadersдёӯжҸҗеҲ°зҡ„пјҢ32дҪҚPEе’Ң64дҪҚPEе·®еҲ«е°ұеңЁдәҺPE Optional HeaderпјҢдё»иҰҒдҪ“зҺ°еңЁдёӨж–№йқў

  1. жҲҗе‘ҳдёӘж•°пјҡ32дҪҚOptionalHeaderжңү31дёӘжҲҗе‘ҳпјҢ64дҪҚOptionalHeaderжңү30дёӘжҲҗе‘ҳпјҢ32дҪҚOptionalHeaderдёӯеӨҡзҡ„жҲҗе‘ҳжҳҜBaseOfDataпјҢе®ғжҳҜDWORDзұ»еһӢпјҢиЎЁзӨәзӣёеҜ№дәҺdataиҠӮзҡ„RVA
  2. дёҖдәӣжҲҗе‘ҳзҡ„зұ»еһӢпјҡжңү5дёӘжҲҗе‘ҳеңЁ32дҪҚOptionalHeaderдёӯжҳҜDWORDзұ»еһӢпјҢеңЁ64дҪҚOptionalHeaderдёӯжҳҜULONGLONGзұ»еһӢпјҢеҲҶеҲ«жҳҜпјҡImageBaseгҖҒSizeOfStackReserveгҖҒSizeOfStackCommitгҖҒSizeOfHeapReserveгҖҒSizeOfHeapCommit

з„¶еҗҺеҲҶеҲ«зңӢдёҖдёӢжҜҸдёӘжҲҗе‘ҳ

  • MagicпјҡиҝҷдёӘеӯ—ж®өиЎЁзӨәеҪ“еүҚзҡ„PEж–Үд»¶жҳҜдёҖдёӘ32дҪҚPEпјҢиҝҳжҳҜдёҖдёӘ64дҪҚPEпјҢеҖјдёә0x10BиЎЁзӨәиҝҷжҳҜдёҖдёӘ32дҪҚPEпјҢеҖјдёә0x20BиЎЁзӨәиҝҷжҳҜдёҖдёӘ64дҪҚPEпјҢIMAGE_FILE_HEADER.MachineжҳҜиў«Windows PE LoaderеҝҪз•Ҙзҡ„
  • MajorLinkerVersionе’ҢMinorLinkerVersionпјҡиҝҷдёӨдёӘеӯ—ж®өиЎЁзӨәй“ҫжҺҘеҷЁзҡ„дё»зүҲжң¬еҸ·е’Ңж¬ЎзүҲжң¬еҸ·
  • SizeOfCodeпјҡиҝҷдёӘеӯ—ж®өиЎЁзӨәtextж®өпјҲд»Јз Ғж®өпјүзҡ„еӨ§е°ҸпјҢеҰӮжһңжңүеӨҡдёӘд»Јз Ғж®өпјҢиЎЁзӨәе…ЁйғЁд»Јз Ғж®өеӨ§е°Ҹзҡ„жҖ»е’Ң
  • SizeOfInitializedDataпјҡиҝҷдёӘеӯ—ж®өиЎЁзӨәdataж®өпјҲеҲқе§ӢеҢ–ж•°жҚ®ж®өпјүзҡ„еӨ§е°ҸпјҢеҰӮжһңжңүеӨҡдёӘеҲқе§ӢеҢ–ж•°жҚ®ж®өпјҢиЎЁзӨәе…ЁйғЁеӨ§е°Ҹзҡ„жҖ»е’Ң
  • SizeOfUninitializedDataпјҡиҝҷдёӘеӯ—ж®өиЎЁзӨәbssж®өпјҲжңӘеҲқе§ӢеҢ–ж•°жҚ®ж®өпјүзҡ„еӨ§е°ҸпјҢеҰӮжһңжңүеӨҡдёӘжңӘеҲқе§ӢеҢ–ж•°жҚ®ж®өпјҢиЎЁзӨәе…ЁйғЁеӨ§е°Ҹзҡ„жҖ»е’Ң
  • AddressOfEntryPointпјҡзЁӢеәҸиў«иҪҪе…ҘеҶ…еӯҳж—¶е…ҘеҸЈзӮ№зҡ„RVAпјҢеҪ“иҪҪе…ҘPEж–Үд»¶ж—¶пјҢе…ҘеҸЈзӮ№жҳҜзЁӢеәҸиө·е§Ӣең°еқҖзҡ„RVAпјҢеҪ“иҪҪе…Ҙй©ұеҠЁзЁӢеәҸж—¶пјҢе…ҘеҸЈзӮ№жҳҜеҲқе§ӢеҢ–еҮҪж•°зҡ„RVAпјҢеҜ№дәҺDLLжқҘиҜҙпјҢе…ҘеҸЈзӮ№жҳҜеҸҜйҖүзҡ„пјҢеҪ“жІЎжңүе…ҘеҸЈж—¶пјҢиҝҷдёӘеӯ—ж®өзҡ„еҖјдёә0
  • BaseOfCodeпјҡPEж–Үд»¶иў«иҪҪе…ҘеҶ…еӯҳеҗҺпјҢtextж®өиө·е§Ӣең°еқҖзҡ„RVA
  • BaseOfDataпјҡеҸӘеӯҳеңЁ32дҪҚPEдёӯпјҢPEж–Үд»¶иў«иҪҪе…ҘеҶ…еӯҳеҗҺпјҢdataж®өиө·е§Ӣең°еқҖзҡ„RVA
  • ImageBaseпјҡиҝҷдёӘеӯ—ж®өиЎЁзӨәPEж–Үд»¶иў«иҪҪе…ҘеҶ…еӯҳеҗҺз¬¬дёҖдёӘеӯ—иҠӮзҡ„ең°еқҖпјҢиҝҷдёӘең°еқҖеҝ…йЎ»жҳҜ64Kзҡ„ж•ҙж•°еҖҚпјҢе®һйҷ…дёҠз”ұдәҺKASLRзӯүеҶ…еӯҳдҝқжҠӨжҺӘж–ҪпјҢеҮ д№ҺдёҚдјҡдҪҝз”ЁиҝҷдёӘең°еқҖдҪңдёәPEзҡ„еҶ…еӯҳеҹәеқҖпјҢPE LoaderдјҡйҖүжӢ©дёҖдёӘжңӘдҪҝз”Ёзҡ„еҶ…еӯҳең°еқҖдҪңдёәPEзҡ„еҶ…еӯҳеҹәеқҖпјҢз„¶еҗҺејҖе§Ӣең°еқҖдҝ®еӨҚпјҢдҝ®ж”№жүҖжңүдҪҝз”ЁиҝҷдёӘең°еқҖдҪңдёәеёёйҮҸзҡ„ең°еқҖпјҢиҝҷдёӘиҝҮзЁӢд№ҹеҸ«йҮҚе®ҡдҪҚпјҢиҝҷдәӣеёёйҮҸдҪҚдәҺдёҖдёӘзү№ж®Ҡзҡ„иҠӮпјҢз§°дёәrelocиҠӮпјҲйҮҚе®ҡдҪҚиҠӮпјү
  • SectionAlignmentпјҡиҝҷдёӘеӯ—ж®өиЎЁзӨәеҶ…еӯҳдёӯиҠӮеҜ№йҪҗзҡ„жңҖе°ҸеҚ•дҪҚпјҢй»ҳи®ӨеҖјжҳҜзү№е®ҡжһ¶жһ„пјҲx86жҲ–x86-64пјүзҡ„еҶ…еӯҳйЎөеӨ§е°ҸпјҢеҫ®иҪҜи§„е®ҡе®ғзҡ„еҖјдёҚиғҪе°ҸдәҺFileAlignment
  • FileAlignmentпјҡиҝҷдёӘеӯ—ж®өиЎЁзӨәж–Үд»¶дёӯиҠӮеҜ№йҪҗзҡ„жңҖе°ҸеҚ•дҪҚпјҢдёҚи¶ізҡ„еЎ«е……0пјҢеҫ®иҪҜи§„е®ҡе®ғзҡ„еҖјеә”иҜҘжҳҜ512еҲ°64 * 1024д№Ӣй—ҙ2зҡ„ж•ҙж¬Ўе№ӮпјҢеҰӮжһңSectionAlignmentзҡ„еҖје°ҸдәҺеҶ…еӯҳйЎөзҡ„еӨ§е°ҸпјҢйӮЈд№ҲFileAlignmentзҡ„еҖје’ҢSectionAlignmentзҡ„еҖјеҝ…йЎ»дёҖиҮҙ
  • MajorOperatingSystemVersionгҖҒMinorOperatingSystemVersionгҖҒMajorImageVersionгҖҒMinorImageVersionгҖҒMajorSubsystemVersionгҖҒMinorSubsystemVersionпјҡMajorOperatingSystemVersionе’ҢMinorOperatingSystemVersionжҢҮе®ҡдәҶжүҖйңҖж“ҚдҪңзі»з»ҹзҡ„дё»зүҲжң¬еҸ·е’Ңж¬ЎзүҲжң¬еҸ·пјҢеҗҺеӣӣдёӘеӯ—ж®өжҢҮе®ҡдәҶPEж–Үд»¶зҡ„дё»зүҲжң¬еҸ·е’Ңж¬ЎзүҲжң¬еҸ·пјҢеӯҗзі»з»ҹзҡ„дё»зүҲжң¬еҸ·е’Ңж¬ЎзүҲжң¬еҸ·
  • Win32VersionValueпјҡдёҖдёӘдҝқз•ҷеӯ—ж®өпјҢеҖјдёә0
  • SizeOfImageпјҡиҝҷдёӘеӯ—ж®өиЎЁзӨәPEж–Үд»¶зҡ„еӨ§е°ҸпјҲеҢ…еҗ«е…ЁйғЁзҡ„еӨҙпјүпјҢеҚ•дҪҚжҳҜеӯ—иҠӮпјҢеҪ“PEж–Үд»¶иў«иҪҪе…ҘеҲ°еҶ…еӯҳж—¶пјҢPE LoaderдјҡдҪҝз”ЁиҝҷдёӘеӯ—ж®өзҡ„еҖјпјҢжүҖд»ҘеҖјйңҖиҰҒжҳҜSectionAlignmentзҡ„ж•ҙж•°еҖҚ
  • SizeOfHeadersпјҡиҝҷдёӘеӯ—ж®өиЎЁзӨәе…ЁйғЁеӨҙзҡ„еӨ§е°ҸпјҢеҢ…жӢ¬д»ҺDOS Headerзҡ„з¬¬дёҖдёӘеӯ—иҠӮеҲ°Section Headerзҡ„жңҖеҗҺдёҖдёӘеӯ—иҠӮ
  • CheckSumпјҡPEж–Үд»¶зҡ„ж ЎйӘҢеҖјпјҢеҪ“PEж–Үд»¶иў«иҪҪе…ҘеҲ°еҶ…еӯҳж—¶з”ЁеҲ°
  • SubsystemпјҡиҝҷдёӘеӯ—ж®өжҢҮе®ҡдәҶиҝҗиЎҢеҪ“еүҚPEж–Үд»¶йңҖиҰҒзҡ„Windowsеӯҗзі»з»ҹпјҲеҰӮжһңжңүзҡ„иҜқпјүпјҢе®Ңж•ҙзҡ„еҖјеҸҜд»ҘеҸӮиҖғпјҡhttps://learn.microsoft.com/en-us/windows/win32/debug/pe-format
  • DLLCharacteristicsпјҡиҝҷдёӘеӯ—ж®өжҢҮе®ҡдәҶPEж–Үд»¶зҡ„дёҖдәӣзү№еҫҒпјҢжҜ”еҰӮжҳҜеҗҰе…је®№DEPжңәеҲ¶гҖҒеҶҚжҜ”еҰӮиғҪеҗҰеңЁиҝҗиЎҢж—¶иҝӣиЎҢйҮҚе®ҡдҪҚпјҢе®Ңж•ҙзҡ„еҖјеҸҜд»ҘеҸӮиҖғпјҡhttps://learn.microsoft.com/en-us/windows/win32/debug/pe-format
